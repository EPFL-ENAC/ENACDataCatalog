How does this platform works ?

ENAC data catalog regroups data used by the ENAC scientists in an unified page. This platform does not store any data itself but makes datasets available by harvesting their metadata from external sources. The software is build using CKAN, an open source data management system.

Which datasets are available ?

Currently, the following datasets are harvested by the platform

  • MeteoSwiss
    The following datasets coming from the Federal Office of Meteorology and Climatology are streamlined
    • Albedo
    • Clear Sky Index
    • Precipitation
    • Radiation
    • Sunshine Duration
    • Temperature

  • Infoscience

    The EPFL Infoscience repository is harvested and all publications marked as ENAC and containing datasets are liked here

Is there an API ?

Yes ! The base url is here and all the functionalities are listed in the Official CKAN API documentation

How to add a dataset ?

If you want your dataset to be made available on this platform, it suffices to publish them on Infoscience. It will be automatically harvested and will appear shortly. If this is not the case do not hesitate to contact us.
